# 实现：

1. 一个命令行程序
2. 需要可以添加启动option来修改交互的以太坊节点，如果没有配置需要给与错误提示并退出
3. 需要提供以下jsonRpc接口：
- 根据区块高度参数(使用Hex编码)或区块label(latest, safe, finalized)参数查询区块链节点，并返回查询到的区块数据
- 根据账户地址参数，返回此地址在2h内发生的所有交易(发送、接收都包含)
- 根据提供的账户地址，向此地址发起一笔转账交易，在交易确认后返回交易信息。
- 无参数，返回最新区块高度
4. 请求默认在5秒内返回，否则返回超时错误，超时时间可以使用启动参数配置
5. 优雅退出，处理完所有请求后才可以退出，在退出时不再接收新的请求
